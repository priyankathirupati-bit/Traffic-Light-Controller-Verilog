# Traffic Light Controller using Verilog HDL

## Overview
This project implements a Traffic Light Controller using a Finite State Machine (FSM) in Verilog HDL.

## Tools
- Verilog HDL
- Xilinx Vivado

## Features
- FSM-based design
- Four traffic light states
- Functional simulation
- RTL schematic

## Files
- traffic_light.v
- traffic_light_tb.v

## Output
Road A: Green → Yellow → Red

Road B: Red → Green → Yellow
